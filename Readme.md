# Конвертер единиц (Лаба 1, Swift/iOS)

Конвертер единиц измерения и валют: 3 категории (расстояние, вес, валюта), по 3 единицы в каждой. Своя цифровая клавиатура, два поля (исходное и результат) без системной клавиатуры. Портрет: блок данных сверху, клавиатура снизу. Альбом: данные слева, клавиатура справа.

## Как открыть

1. В Xcode: **File → New → Project → App** (iOS, Swift, SwiftUI).
2. Сохрани проект в папку `mobile-ios`, имя например `ConverterApp`.
3. Удали сгенерированные файлы в группе проекта и добавь в проект папку `ConverterApp` (все `.swift` файлы из неё).

Либо создай новый iOS App и скопируй содержимое папки `ConverterApp` в группу с исходниками проекта.

## Подпись приложения (п. 1.2)

Итоговый билд для релиза должен подписываться личным сертификатом:

1. В Xcode открой проект → выбери таргет приложения.
2. Вкладка **Signing & Capabilities**.
3. Включи **Automatically manage signing**, выбери **Team** (твой Apple ID или команда).
4. Для релиза (Archive) подпись выполнится автоматически при сборке.

Так система проверяет, что приложение не изменено после сборки и от кого оно установлено.

## Build flavour «Premium» (п. 2.1)

В премиум-варианте доступны: кнопка обмена местами исходного и преобразованного значения (и единиц) и кнопки копирования в буфер обмена у каждого поля.

Как включить Premium в сборке:

1. **Project → Build Settings** для таргета приложения.
2. В поиске введи **Swift Compiler - Custom Flags** (или **Other Swift Flags**).
3. В **Active Compilation Conditions** (или **Other Swift Flags**) для нужной конфигурации (например Debug или отдельная схема) добавь: `PREMIUM`.

Либо создай вторую конфигурацию (например **Premium**) и в ней задай флаг `PREMIUM`. Сборка без этого флага — обычная версия, с флагом — premium (кнопки «поменять местами» и «копировать»).

## Структура

- `ConverterApp.swift` — точка входа.
- `Converter.swift` — категории, единицы, конвертация.
- `DataView.swift` — блок с полями и выбором единиц (и премиум-кнопки при `PREMIUM`).
- `KeyboardView.swift` — клавиатура калькулятора (цифры, точка, сброс, backspace).
- `ContentView.swift` — компоновка: портрет (VStack), альбом (HStack), логика ввода и вызов конвертера.

Поля только для отображения, ввод только с нашей клавиатуры.
